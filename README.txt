## INFORMATION ##

Moderation Notice (1.0) for MyBB 1.6
Created by: Starpaul20
Copyright: ©2011
License: GPL

Displays a notice to users if posts/threads/attachments are being moderated for them. Messages are displayed in a yellow box above the posting forms.

This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Activate"
4. Enjoy!


## UPDATING ##

Any beta version should be reinstalled.
